# VirtualEnv
Per creare una VirtualEnv su Linux con MiniConda da riga di comando:
conda create --name DeskApp python=3.10

Poi per attivare/disattivare la VirtualEnv utilizzare i comandi:
conda activate DeskApp
conda deactivate
